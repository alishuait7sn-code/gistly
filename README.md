# ⚡ Gistly | The Architect's Lexicon

**Gistly** is a minimalist, high-contrast Command Line Interface (CLI) tool designed for capturing and memorizing the "gist" of complex terms. Built specifically for developers and aspiring System Architects who want to bridge the gap between reading technical documentation and truly retaining core concepts.

---

## ✨ Features

- **Zero Dependencies:** Built entirely with Python's standard library. No `pip install` required.
- **Portable Architecture:** Uses dynamic relative paths. Works instantly on Windows, Mac, or Linux.
- **Searchable Foundation:** Quickly find previously stored terms to refresh your memory.
- **Architectural Mindset:** A "Flat-File" proof-of-concept, ready for future migration to SQLite.
