# ⚡ Gistly | The Digital Lexicon 
### *Capture the Essence, Master the Language*

**Gistly** is a minimalist CLI tool built for one purpose: **Active Retention**. Most developers read documentation, forget 90% of it, and have to search for the same terms again. Gistly breaks that cycle by forcing you to distill complex concepts into a "gist"—a short, personal summary that sticks in your memory.

---

## 🎯 The Core Philosophy / الفلسفة الأساسية

### English
Learning a new language or a new technical field isn't about reading; it’s about **encoding**. Gistly encourages you to:
1. **Define:** Put the term in your own words.
2. **Categorize:** Connect it to a larger mental map.
3. **Apply:** Write a real-world usage example.
By doing this, you aren't just storing data—you are building knowledge.

### بالعربية
تعلم لغة جديدة أو مجال تقني جديد لا يقتصر على القراءة فقط، بل يتعلق بـ **الترميز الذهني**. يشجعك Gistly على:
1. **التعريف:** صياغة المصطلح بأسلوبك الخاص.
2. **التصنيف:** ربط المصطلح بخريطة ذهنية أوسع.
3. **التطبيق:** كتابة مثال واقعي لاستخدام المصطلح.
من خلال القيام بذلك، أنت لا تقوم فقط بتخزين البيانات، بل تقوم ببناء معرفة حقيقية.

## ✨ Features / المميزات

- **Zero Dependencies:** Pure Python. Runs anywhere instantly.
- **Bilingual Support:** Store terms in English, Arabic, or both.
- **Searchable Database:** Find your personal definitions in seconds.
- **Portable JSON:** Your knowledge is stored in a simple file you can carry with you.
